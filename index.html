Dashboard

This project was done through the process of full stack development. First we had to create user stories about what was needed and what would be liked by the users. Then decide on functions were necessary to the program and what features would be neat to have. Next step was to create UML diagrams for classes that needs to be implemented. After the diagrams are created, the next step was to use the process of test driven development to create test and figure out what is the correct output. After the project was complete, the final step was to create java documentation of the functions.

Through out the process of completing this project I learn a bit about how to use Maven and Spring-boot API. The project taught me a lot about the brainstorming and development phase of create a application. It also taught me good programing practices that I should use whether I am working on a project on my own or in a group.

This project was written using Java, Maven, and Spring-boot.

Green Button Initiative

This project executed following the SCRUM methodology. Initially, we engaged in comprehensive discussions with our clients to understand their identity and requirements. Subsequently, armed with this knowledge, we started creating an front-end website for client review. Bi-weekly, we organized showcases to demonstrate our advancements and request feedback for any modifications.

Through out the process of completing this project I learn about how an SCRUM software developer, learned how to use Angular framework along with a few libraries, and it also help solidify my ability to work in a team environment and my ability to use git for version control.

This project was written using Angular and typescript.

Along with features that allows you to add, update, delete data.

Through out the process of completing this project I learn how to develop Restful APIS and how to use post, get, and put. The project taught me the steps needed to create a application from setting up the database to creating the API, to using the API with a website application.

This project was written using html, javascript, sqlite3, and nodejs.
